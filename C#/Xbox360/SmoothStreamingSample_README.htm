Smooth Streaming and PlayReady Sample

Description

This sample demonstrates video playback by using a Smooth Streaming Media Element (SSME) control in the Xbox 360 Application Development Kit (ADK). The player adaptively switches to the best possible bitrate for video, based on available bandwidth. If your Internet connection bandwidth is sufficiently high, you'll experience high-definition video playback of the sample content. The sample uses a live data graph control to illustrate the video download and playback bitrate.

This sample also demonstrates how PlayReady is integrated with SSME. The sample plays two PlayReady-protected videos where the videos are hosted on the PlayReady test server located at http://playready.directtaps.net. The first video, “Super Speedway,” demonstrates video-on-demand (VOD) content that requires a traditional PlayReady license whereby one content key is required to decrypt the entire video. The video uses licenses generated on the fly from http://playready.directtaps.net/pr/svc/rightsmanager.asmx (see http://playready.directtaps.net/pr/doc/customrights/ for more details).

The second video, “To The Limit,” demonstrates a Live TV scenario whereby the content is encrypted with different keys and the keys are rotated at some variable time period. The root license necessary to access the rotated keys is delivered from a PlayReady license server built with the PlayReady Server Software Development Kit (SDK) Premium Edition. The key rotation period for this content is 30 seconds, that is, every 30-second interval of the content is encrypted with a different content key. Each content key is embedded within the content stream (per the PIFF specification used in Smooth Streaming) and the SSME transparently accesses the keys by using the root license. Real world Live TV applications will typically pre-deliver these root licenses before any Live TV content is played. Demonstration of this pre-delivery mechanism using the LicenseAcquirer class is shown by the last control of the sample application. The control for the “To The Limit” video is disabled until the root license pre-delivery is done using the last control.

Root licenses, when used in traditional non-Live TV scenarios, are not complex. However, for Live TV scenarios, how they are used to enforce channel packages and blackout policies is a complex subject. You can find more details in the PlayReady Server SDK Premium Edition documentation. With this video, for the test server to know how to issue the required root license to allow decryption of the content, the custom string "UserAccount:0" must be added to the license challenge string for that content. This is the arbitrary way the test license server is set up to handle root license issuance for this content. Your real world license server will likely have different requirements for what your application must send.

Furthermore, this sample uses Media Transport Control (MTC), which provides a standard playback user interface for use with ADK applications.

 

Controls

Once you run the sample, you can use the remote control, gamepad, or NUI hands to interact with the sample. In order to use NUI hands, ensure that your Kinect Sensor is attached to your development kit, and then wave to engage. The Feedback Control at the bottom right corner of the screen will provide you feedback while you use NUI hands to interact with this page.

 

Once you select a clip, the video playback should start shortly. The sample uses two graphs to illustrate the download and playback bitrate. Press Y on the gamepad to hide these overlay graphs, and press X to return to previous page.

 

While playing a video, you can use NUI hands, gamepad, or remote control to interact with the MTC and to control video playback. To use NUI hands, wave to engage and activate the NUI hand cursor, and then move your hand to the top of screen. This will open the MTC UI.

1280 × 720, VC1 Video, WMA Pro Audio, 2 min

DRM protected. Uses a persistent root license and embedded license leaf licenses within the Smooth Streaming content

 

·         “Super Speedway”

Manifest URI: http://playready.directtaps.net/smoothstreaming/SSWSS720H264PR/SuperSpeedway_720.ism/Manifest

License URI: http://playready.directtaps.net/pr/svc/rightsmanager.asmx?PlayRight=1&FirstPlayExpiration=10

1280 × 720, H.264 Video, AAC-L Audio, 2 min

DRM protected. Uses a persistent license

The license expires 10 seconds after the first playback

Note: In most applications, the thumbnail images used by MTC are downloaded from a server. The streaming servers used by this sample do not host any thumbnail images; therefore, the sample loads some default images to show how thumbnail images are to be passed to MTC.

Known Issues:

The sample performance will be very low and may crash if run in Managed Debug mode. Please either user Debug or Release modes.

This sample will not run properly if built using the Submission build configuration.

The development kit running the sample must be set to 720p output resolution, otherwise some clipping may occur.

Enabling the bitrate graphs may have a small performance hit on the NUI hand cursor. Hiding these graphs by pressing the Y button will remove this performance hit.

In order to play the "Super Speedway" clip, you must download Optional Media Update 3 from Xbox LIVE Marketplace. See the ADK Release Notes for instructions. This limitation will be addressed in a future release of the ADK.
