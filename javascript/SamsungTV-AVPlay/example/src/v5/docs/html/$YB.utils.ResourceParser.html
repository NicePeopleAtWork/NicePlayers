Class: ResourceParser

$YB.utils.ResourceParser

new ResourceParser(viewManager)

Youbora utils.ResourceParser will parse the precise CDN of the resource if options.parseHLS or options.parseCDNNodeHost are true. CDNs will be parsed in the order defined in $YB.utils.ResourceParser.cdnsAvailable, modify context list to modify order.
