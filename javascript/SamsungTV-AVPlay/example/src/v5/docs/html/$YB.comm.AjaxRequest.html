Class: AjaxRequest

AjaxRequest

new AjaxRequest(host, serviceopt, paramsopt, optionsopt)

AjaxRequest will generate the call URL. See createServiceUrl.
Parameters:
Name Type Attributes Description
host string URL of the request. ie: nqs.nice264.com
service string <optional>
 Name of the service. ie '/start'
params string | Object <optional>
 String of params. Skip '?' at start. ie: 'system=nicetv&user=user'.
options Object <optional>
 Object with custom options.
Properties
Name Type Attributes Default Description
method string <optional>
 GET Specifies the method of the request ie: "GET", "POST", "HEAD".
requestHeaders string <optional>
 A literal with options of requestHeaders. ie: {header: value}.
retryAfter number <optional>
 5000 Time in ms before sending a failed request again. 0 to disable.
maxRetries number <optional>
 3 Max number of retries. 0 to disable.
