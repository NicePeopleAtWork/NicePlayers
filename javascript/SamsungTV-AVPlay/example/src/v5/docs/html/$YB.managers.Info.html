Class: Info

$YB.managers.Info

new Info(plugin, options)

Info API is in charge to retrieve information from OPTIONS, plugins or adnalyzers using getXXX functions (ie: getResource).
Parameters:
Name Type Description
plugin $YB.plugins.Generic The plugin from where it was called.
options object Literal object that will be uset to initialize $YB.data.Options.
