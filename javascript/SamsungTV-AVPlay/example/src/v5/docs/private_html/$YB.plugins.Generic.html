Class: Generic

$YB.plugins.Generic

new Generic()

This class is the generic plugin from which specifics plugins will extend. Internally, it coordinates a number of inner components like AdsApi, VideoApi, Communications, Chrono, ResourceParser, Resumer, ConcurrencyChecker...
