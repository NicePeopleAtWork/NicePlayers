Class: Communication

$YB.comm.Communication

(private) new Communication(plugin)

Youbora Communication implements the last abstraction layer against NQS requests. Internally, Communication implements an array of $YB.comm.AjaxRequest objects, executing one after another. All requests will be blocked until a first /data call is made, before context, any request sent will be queued.
